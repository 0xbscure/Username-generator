# Username generator
Generating user list from username you have been collected, useful for userenum [Kerbrute](https://github.com/ropnop/kerbrute)

### Usage
```
usage: generate.py [-h] -w wordlist [-u]

Generate Username from user you have been collected

optional arguments:
  -h, --help            show this help message and exit
  -w wordlist, --wordlist wordlist
                        Specify path to the wordlist
```
